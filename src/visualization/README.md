# Visualization Module

This folder contains utilities for saving visual debugging outputs.

## Purpose

Save images and visual proof from different pipeline stages.

## Files

- `frame_saver.py`
  - saves selected frames or debug images to disk

## Current Use

This module is used to save debug frames and pose visualization images.

## Future Use

Later modules may use this folder for:

- trajectory plots
- key-frame overlays
- event-frame images
- final report visuals

## Status

Basic visualization support is working.
